# 1-Minute Demo Script — EventBridge

---

**[0:00 — 0:10] Hook**

"Organizing a live event usually starts with chaos — vague ideas, scattered vendor contacts, group chats, and unclear responsibilities. This is EventBridge."

---

**[0:10 — 0:25] Input**

"I'm going to type a rough event idea — an underground music and art event in Vancouver for 150 people with sound, lights, visuals, food, security, and a community-focused vibe. Budget around $8,000."

*[Type the event idea into the textarea and click Generate Event Blueprint]*

---

**[0:25 — 0:40] Blueprint + Stack + Risks**

"The app turns that into a structured event blueprint — event name, type, audience size, budget, vibe, and a summary. Then it identifies the full event stack with priority levels: must-have, important, optional. And it surfaces operational risks specific to this event."

*[Scroll through blueprint, categories, risks]*

---

**[0:40 — 0:55] Vendors + Outreach**

"Then it matches service providers based on event type and vibe — with fit scores and why they were matched. And for any vendor, I can generate a personalised outreach message in one click."

*[Click Generate Outreach Message on Bassline Audio — show the message appear]*

---

**[0:55 — 1:00] Closer**

"So instead of spending hours turning an idea into an execution plan, organizers go from idea to vendor-ready brief in under 60 seconds. That's EventBridge."

---

## Tips

- Type a real custom event (not the placeholder) so AI generates a fresh result
- Let the blueprint finish loading before scrolling — pause there for effect
- Click outreach on the vendor that best matches your event type
- Have the copy button ready to show the full workflow
